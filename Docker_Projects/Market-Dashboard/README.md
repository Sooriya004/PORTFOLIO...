# Market Dashboard (Dockerized)

Multi-market financial dashboard displaying live prices.

## Features
- Bitcoin
- Gold & Silver
- Nifty 50
- Sensex
- Nasdaq
- S&P 500

## API USED
- Binance API
- Yfinance

## Tech Stack
- Flask
- Docker
- Docker Compose
- Redis

## Run Locally

docker compose up

Then open:
http://localhost:5000
